Detailed Description

Class TimeValidator

Menangani validasi waktu kehadiran, termasuk pengecekan hari kerja, jendela waktu masuk/pulang, serta pengambilan jadwal kerja karyawan dari berbagai tingkatan (shift, jadwal rutin, atau default).
