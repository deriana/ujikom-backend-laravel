Detailed Description

Class PayrollController

Controller untuk mengelola sistem penggajian (Payroll) karyawan, mencakup pembuatan data gaji, pembaruan, finalisasi (pembayaran), pembatalan (void), serta pembuatan dan pengunduhan slip gaji.
